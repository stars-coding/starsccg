# acm-template-pro-generator

> ACM 代码生成器
>
> 作者：stars
>
> 一个可以在线个性化定制代码生成器的平台

可以通过命令行交互式输入的方式动态生成想要的项目代码

## 使用说明

执行项目根目录下的脚本文件：

```
generator <命令> <选项参数>
```

示例命令：

```
generator generate -l
```
